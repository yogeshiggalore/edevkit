Target reset

    

    

      Three different reset paths. System reset is the standard ARM SYSRESETREQ.
      Reset & halt sets DEMCR.VC_CORERESET first so the core stops at the reset vector
      (useful for debugging boot code). Nordic CTRL-AP reset writes 1 to the CTRL-AP RESET
      register and works even when APPROTECT blocks the AHB-AP.
