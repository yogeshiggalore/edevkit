What's inside

six pillars Every edevkit is one device, six bench tools.

Debug probe

CMSIS-DAP v2 over USB. SWD + JTAG up to 25 MHz. Works with pyOCD, OpenOCD, probe-rs, and GDB out of the box.

SWO trace

ITM/TPIU output decoded and streamed to host over the same USB endpoint as the debug session — no extra wires.

Two UART bridges

One for the target's log console, one for protocol testing. Independent voltage domains so 3.3 V and 1.8 V targets coexist.

I²C tester

Master, slave, or sniffer at any address up to 3.4 MHz. Software-defined; reconfigure live without re-flashing.

SPI / DSPI / QSPI tester

Master / slave / sniffer in every CPHA/CPOL mode. Up to 50 MHz QSPI master via PIO acceleration.

Device emulator

YAML-driven. Plays the role of a real SPI flash chip / I²C sensor / UART AT-modem so you can develop the host side without the actual peripheral.
