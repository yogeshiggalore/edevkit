How to use this book

1

Read the chapter

Each chapter is self-contained: concept exposition, byte-level breakdowns, diagrams. Theory chapters in Part I have no sample — start there if you've never enumerated a USB device.

2

Build the paired sample

From Part II onward every chapter has a sample under firmware/samples/usb/usb_NN_*/. Build, flash, plug in.

3

Verify on your OS

Each chapter ships exact verification commands for Linux, macOS, and Windows. If they don't all work, the chapter fails its quality bar — file an issue.

4

Hit the pitfalls section

The mistakes other people made writing this code, condensed. Read these before debugging — they save hours.
